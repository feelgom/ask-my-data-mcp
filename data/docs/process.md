# 시공 판단 프로세스

## 개요
시공 판단은 특정 BIM 요소가 현장에 실제로 설치되었는지 자동으로 판정하는 과정입니다.
설계 데이터(BIM Mesh)와 현장 스캔 데이터(Point Cloud)를 비교해 판단합니다.

## 단계
1. 현장을 360 파노라마로 촬영하고 Point Cloud를 생성합니다.
2. Point Cloud와 BIM Mesh를 ICP로 정렬해 좌표계를 맞춥니다.
3. 각 BIM 요소 위치에 대응하는 점군이 충분히 존재하는지 확인합니다.
4. 점군 밀도와 형상 일치도를 기준으로 설치 여부를 판정합니다.

## 정확도
판단 정확도는 실제 설치 여부와 알고리즘 판정이 일치하는 비율로 측정합니다.
정렬 오차가 크거나 점군이 희박하면 오탐이 늘어날 수 있어, 전처리와 정렬 품질이 중요합니다.
